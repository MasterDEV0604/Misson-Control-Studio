# Product Requirements

## Core operator tasks
- Open a scenario and inspect validation state.
- Review platform health, route state, and mission objectives.
- Change track focus and filter the view.
- Edit route points and add operator notes.
- Replay a scenario timeline and export a debrief.
- Restore the previous workspace after restart.

## Non-functional goals
- Fast UI startup for sample scenarios.
- Clear event journal for auditability.
- Sanitary, fictional scenario data suitable for public display.
- Large-screen workstation layout appropriate for engineering demos.
